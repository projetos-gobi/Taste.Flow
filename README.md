## Taste.Flow

Monorepo com Backend (.NET) e Frontend (Next.js/TypeScript).

### Estrutura
- `Backend/`: APIs em .NET 8
- `Frontend/`: app Next.js (Node)

### Requisitos
- .NET 8 SDK
- Node 18+

### Como rodar
Backend:
```bash
cd Backend/TasteFlow.Api
dotnet run
```

Frontend:
```bash
cd Frontend
npm install
npm run dev
```
